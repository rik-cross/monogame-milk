Class Sprite

Namespace
milk.Core
Assembly
MonoGameECS.dll

A sprite is added to a SpriteComponent, and is a collection of textures, along with some additional data. A sprite with more than one texture automatically becomes an animated sprite.

duration double

The total time to display the animation sequence (default = 0).

offset Vector2?

The distance between the entity position and the sprite (default = none).

scale Vector2?

The scale factor for the textures (default = 1 - no scale factor).

loop bool

Loop the textures (default = true).

hue Color?

The hue to recolor the sprite (default = white / no recolor).

flipH bool

Horizontally flip the textures (default = false).

flipV bool

Vertically flip the textures (default = false).
