Parameters

texture Texture2D

The texture to display.

state string

The state to associate the texture with (default = "default" state).

resizeToEntity bool

Resize the texture to match the entity's TransformComponent dimensions (default = false).

offset Vector2?

The (x, y) position to offset the texture by (default = null - (0, 0)).

scale Vector2?

The (w, h) scale factor for the texture (default = null - use original size).

hue Color?

Add a color filter to the texture (default = null - Color.White (no filter)).

alpha float

The texture transparency (default = 1.0f - fully visible).

flipH bool

Horizontally flip the texture (default = false).

flipV bool

Vertically flip the texture (default = false).
