Parameters

numberOfSlots int

The number of inventory slots (default = 8).

allowInputWrapping bool

Allows cyclical movement between inventory slots (default = false).

position Vector2?

The (x, y) position of the inventory (default = null - (0, 0)).

anchor Anchor

The anchor to set the position against (default = Anchor.TopLeft).

slotSize Vector2?

The (w, h) slot size in pixels (default = null - (64, 64)).

slotsPerRow int?

The number of slots in a row (default = null - all on a single row).

margin int

The gap between slots (default = 8).

alpha float

The opacity of the inventory, between 0 anf 1 (default = 1 - fully opaque).

alphaInactive float

The opacity of the inventory when inactive, between 0 anf 1 (default = 1 - fully opaque).

primaryColor Color?

The main color used to draw the inventory (default = null - Color.White).

primaryColorInactive Color?

The main color used to draw the inventory when inactive (default = null - Color.LightGray).

secondaryColor Color?

The accent color used to draw the inventory (default = null - Color.Black).

secondaryColorInactive Color?

The accent color used to draw the inventory when inactive (default = null - Color.DarkGray).

font SpriteFont

The font used to draw slot names and numbers (default = null - FontSmall).

customDrawMethod Action<InventoryComponent>

Specifies a method to override the drawing of the inventory (default = null).

visible bool

Sets the visibility of the inventory (default = true).

active bool

Sets the ability to interact with the component (default = true).

inputActions Dictionary<Keys, Action>

Maps Keys to Actions (default = null - no actions mapped).
